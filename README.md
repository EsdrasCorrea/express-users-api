# TaskFlow API

TaskFlow API is a backend project built with Node.js and Express.js.

The goal of this project is to practice backend fundamentals using a real-world structure, including routes, controllers, services, middlewares, validation, error handling, and later data persistence.

## Tech Stack

- Node.js
- Express.js
- Nodemon

## Features

- List users
- Get user by ID
- Create user
- Delete user
- Request logger middleware
- User validation middleware
- Global error handler
